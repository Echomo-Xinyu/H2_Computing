Create Form

The create form command text file should adhere to the following rules:

  1. The first line of the text file should be the name of the form. The name should only contain alpha numeric characters with the exception of space between words.
  2. All of the following lines should contain information of the form elements, one line per element
  3. Each of the following line should follow this format <field_name>,<field_type>,<max_length(optional)>. (Note that the default max_length will always be 1000.)
  4. The field name may have spaces ONLY INBETWEEN WORDS.
  5. The field type can only be one of the following:
    • text
    • textarea
    • file
    • button
    • number (maxlength ignored)
    • password
  6. If field type is 'button', then that line has a special third parameter specifying the type like such: Press Me,button,reset. These are the following button types
    • submit (only in the last line)
    • reset
    Hence, the only additional button you may use is the reset button.
  7. The last line is a special line reserved for the 'submit' button. In this line write, <field_label>,<url_to_submit_to>

A few sample command text files can be found in the root folder for your reference.
